# 02 - HTML Elements and Tags

## Introduction

In the previous lesson, you created your first HTML page.

Now, let's understand what HTML **tags** and **elements** are. These are the building blocks of every HTML webpage.

---

## What is an HTML Tag?

A tag tells the browser what type of content you are creating.

For example:

```html
<h1>
```

This is an HTML tag.

Another example:

```html
<p>
``` 

This is also an HTML tag.

---

## What is an HTML Element?

An HTML element is everything from the opening tag to the closing tag, including the content between them.

Example:

```html
<h1>Welcome to HTML</h1>
```

This entire line is one HTML element.

Another example:

```html
<p>This is my first paragraph.</p>
```

This is also an HTML element.

---

## Example

```html
<h1>Learning HTML</h1>

<p>HTML is easy to learn with practice.</p>
```

The browser displays a heading followed by a paragraph.

---

## Try It Yourself

Open the `index.html` file and:

* Change the heading text.
* Change the paragraph text.
* Save the file.
* Refresh your browser to see the changes.

Try changing the text several times until you're comfortable editing HTML.
