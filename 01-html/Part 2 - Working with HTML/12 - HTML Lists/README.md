# 12 - HTML Lists

## Introduction

Lists are used to display a group of related items.

HTML provides different types of lists. In this lesson, you'll learn about **unordered lists** and **ordered lists**.

---

## Unordered List

An unordered list displays items with bullet points.

The `<ul>` tag creates the list, and each item is added using the `<li>` tag.

### Syntax

```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
```

### Example

```html
<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
</ul>
```

---

## Ordered List

An ordered list displays items with numbers.

The `<ol>` tag creates the list, and each item is added using the `<li>` tag.

### Syntax

```html
<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>
```

### Example

```html
<ol>
    <li>Wake Up</li>
    <li>Have Breakfast</li>
    <li>Go to School</li>
</ol>
```

---

## Output

The webpage displays:

**Unordered List**

* Apple
* Banana
* Orange

**Ordered List**

1. Wake Up
2. Have Breakfast
3. Go to School

## What You Learned

After completing this lesson, you should know:

* The `<ul>` tag creates an unordered list.
* The `<ol>` tag creates an ordered list.
* The `<li>` tag is used to add list items.
* Lists help organize related information in a clear and readable way.
