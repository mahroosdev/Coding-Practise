# Final HTML Website

## Introduction

This project is the final practical exercise for the HTML section of my web development learning journey.

The website was built using only HTML5 and brings together the topics covered throughout the course. The main purpose of this project is to practice creating a complete multi-page website while using proper HTML structure and semantic elements.

No CSS or JavaScript has been used in this project.

---

## Project Pages

- Home
- About
- Services
- Gallery
- Contact

Each page focuses on different HTML concepts while working together as one complete website.

## Learning Purpose

This project was created to practice building a complete website using only HTML before moving on to CSS.

The main goal was to understand how webpages are structured and how different HTML elements work together in a real project.

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

No installation or additional software is required.
