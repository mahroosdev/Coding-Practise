HTML Meta Tags

Meta tags provide information about a webpage.

Where Are Meta Tags Used?

Meta Tag Where It Is Used
<meta charset="UTF-8"> Browser uses it to display characters correctly.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> Browser uses it to make the page responsive on phones and tablets.
<meta name="description" content="..."> Search engines may show it in search results.
<meta name="keywords" content="..."> Historically used by search engines. Today, it is largely ignored by major search engines.
<meta name="author" content="Your Name"> Stores information about the page author.
