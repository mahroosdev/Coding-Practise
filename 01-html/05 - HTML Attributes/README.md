# 05 - HTML Attributes

## Introduction

In the previous lessons, you learned about HTML tags like `<h1>` and `<p>`.

Sometimes, we need to give extra information about an HTML element. We do this by using **attributes**.

---

## What is an HTML Attribute?

An attribute provides extra information about an HTML element.

Attributes are written inside the opening tag.

---

## Syntax

```html
<tagname attribute="value">Content</tagname>
```

---

## Example

```html
<h1 title="Welcome">
    HTML Attributes
</h1>
```

In this example:

* `title` is the attribute.
* `"Welcome"` is the value of the attribute.

If you move your mouse over the heading, you'll see the text **Welcome**.

---

## Another Example

```html
<p title="Learning HTML">
    HTML is easy to learn.
</p>
```

The `title` attribute adds extra information to the paragraph.

---

## Output

When you open the webpage, you'll see a heading and a paragraph.

If you place your mouse over either one, a small tooltip will appear showing the text from the `title` attribute.

---

## Practice

Create:

* One heading with a `title` attribute.
* Two paragraphs with different `title` attributes.

Move your mouse over each element to see the tooltip.

---

## What You Learned

After completing this lesson, you should know:

* What an HTML attribute is.
* Attributes provide extra information about an element.
* Attributes are written inside the opening tag.
* Every attribute has a name and a value.
