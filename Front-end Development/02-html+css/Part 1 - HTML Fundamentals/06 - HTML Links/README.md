# 06 - HTML Links

## Introduction

Links allow users to move from one webpage to another.

In HTML, links are created using the `<a>` tag. The letter **a** stands for **anchor**.

---

## What is an HTML Link?

An HTML link connects one webpage to another.

When a user clicks a link, the browser opens the page specified in the link.

---

## Syntax

```html
<a href="https://example.com">Link Text</a>
``` 

* `<a>` creates the link.
* `href` specifies the destination.
* The text between the tags is what the user clicks.

---

## Example

```html
<a href="https://developer.mozilla.org/en-US/">
    Visit MDN Web Docs
</a>
```

When the link is clicked, the browser opens the MDN Web Docs website.

---

## Output

The webpage displays:

```text
HTML Links

Click the link below to visit the MDN Web Docs website.

Visit MDN Web Docs
```

The text **Visit MDN Web Docs** appears as a clickable link.

---

## What You Learned

After completing this lesson, you should know:

* The `<a>` tag is used to create links.
* The `href` attribute specifies where the link goes.
* The text between `<a>` and `</a>` is the clickable text.
* Clicking a link opens another webpage.
