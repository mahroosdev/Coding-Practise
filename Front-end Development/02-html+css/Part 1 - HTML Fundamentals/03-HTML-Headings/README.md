# 03 - HTML Headings

## Introduction

Headings are used to give titles and subtitles to the content on a webpage.

HTML provides six heading levels, from `<h1>` to `<h6>`.

---

## What are HTML Headings?

HTML headings help organize information on a webpage.

The browser displays each heading with a different size.

* `<h1>` is the largest heading.
* `<h6>` is the smallest heading.

---

## Syntax

```html
<h1>Heading</h1>
``` 

Replace **Heading** with the text you want to display.

---

## Example

```html
<h1>Welcome to HTML</h1>

<h2>About HTML</h2>

<h3>Learning Basics</h3>
```

---

## Output

When you open the example file, you'll see six headings.

The first heading is the largest, and each heading below it becomes slightly smaller.

---

## Practice

Create your own headings using the following text:

```text
My Portfolio

About Me

Skills

Projects

Contact
```

Try changing the text to anything you like and refresh the browser to see the result.

---

## What You Learned

After completing this lesson, you should know:

* HTML has six heading tags.
* Headings are used to organize content.
* `<h1>` is the largest heading.
* `<h6>` is the smallest heading.
* Heading text can be changed to display any title you want.
