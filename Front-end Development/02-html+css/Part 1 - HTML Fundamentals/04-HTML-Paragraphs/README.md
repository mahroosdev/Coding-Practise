# 04 - HTML Paragraphs

## Introduction

Headings are used for titles, but most of the information on a webpage is written using paragraphs.

HTML uses the `<p>` tag to create paragraphs.

---

## What is a Paragraph?

A paragraph is used to display normal text on a webpage.

Every paragraph starts with an opening `<p>` tag and ends with a closing `</p>` tag.
 
---

## Syntax

```html
<p>Your text goes here.</p>
```

---

## Example

```html
<p>This is my first paragraph.</p>

<p>I am learning HTML.</p>
```

---

## Output

The browser displays each paragraph on a new line.

Example:

```text
This is my first paragraph.

I am learning HTML.
```

---

## Practice

Create a webpage with:

* One heading
* Three paragraphs

Use your own text for each paragraph.

For example, you can write about:

* Yourself
* Your hobbies
* Your favorite subject

---

## What You Learned

After completing this lesson, you should know:

* The `<p>` tag is used to create paragraphs.
* Paragraphs are used to display normal text.
* Each paragraph starts with `<p>` and ends with `</p>`.
* You can add multiple paragraphs to a webpage.
