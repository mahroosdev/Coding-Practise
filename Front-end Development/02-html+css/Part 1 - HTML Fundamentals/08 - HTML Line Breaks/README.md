# 08 - HTML Line Breaks

## Introduction

Sometimes, you may want to move text to the next line without creating a new paragraph.

In HTML, you can do this using the `<br>` tag.

---

## What is a Line Break?

A line break moves the text to the next line.

It does not create a new paragraph. Instead, it continues the same paragraph on a new line.

---

## Syntax

```html
<br>
```

Place the `<br>` tag where you want the text to move to the next line.

---

## Example

```html
<p>
    My name is John.<br>
    I am learning HTML.<br>
    I enjoy building websites.
</p>
```

---

## Output

The browser displays:

```text
My name is John.
I am learning HTML.
I enjoy building websites.
```

Each sentence appears on a new line, but they are still part of the same paragraph.

---

## Practice

Create a paragraph about yourself.

Use the `<br>` tag to place each sentence on a new line.

For example:

* Your name
* Your country
* Your favorite hobby

---

## What You Learned

After completing this lesson, you should know:

* The `<br>` tag creates a line break.
* It moves text to the next line.
* It does not create a new paragraph.
* You can use multiple `<br>` tags in the same paragraph when needed.
