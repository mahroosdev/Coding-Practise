# 11 - HTML Text Formatting

## Introduction

HTML provides several tags to format text.

These tags help make important text stand out, show changes, display formulas, and improve the appearance of your webpage.

---

## Bold Text

The `<b>` tag displays text in **bold**.

### Example

```html
<b>This text is bold.</b>
```

---

## Important Text

The `<strong>` tag marks text as important.

### Example

```html
<strong>This text is important.</strong>
```

---

## Italic Text

The `<i>` tag displays text in *italic*.

### Example

```html
<i>This text is italic.</i>
```

---

## Emphasized Text

The `<em>` tag emphasizes text.

### Example

```html
<em>This text is emphasized.</em>
```

---

## Highlighted Text

The `<mark>` tag highlights text.

### Example

```html
<mark>This text is highlighted.</mark>
```

---

## Small Text

The `<small>` tag displays smaller text.

### Example

```html
<small>This text is smaller.</small>
```

---

## Deleted Text

The `<del>` tag displays deleted text with a line through it.

### Example

```html
<del>This text has been deleted.</del>
```

---

## Inserted Text

The `<ins>` tag displays inserted text with an underline.

### Example

```html
<ins>This text has been inserted.</ins>
```

---

## Subscript Text

The `<sub>` tag displays subscript text.

### Example

```html
H<sub>2</sub>O
```

Output:

```text
H₂O
```

---

## Superscript Text

The `<sup>` tag displays superscript text.

### Example

```html
x<sup>2</sup>
```

Output:

```text
x²
```
