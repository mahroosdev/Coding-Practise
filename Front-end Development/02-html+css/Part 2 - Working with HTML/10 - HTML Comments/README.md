# 10 - HTML Comments

## Introduction

As your HTML files become larger, it can be helpful to leave notes in your code.

These notes are called **comments**.

Comments are written for developers. They help explain the code and make it easier to understand later.

---

## What is an HTML Comment?

An HTML comment is a note written inside an HTML file.

The browser ignores comments, so they are **not displayed** on the webpage.

---

## Syntax

```html
<!-- This is a comment -->
```

Everything between `<!--` and `-->` is treated as a comment.

---

## Example

```html
<!-- Main Heading -->
<h1>Welcome to My Website</h1>

<!-- Paragraph -->
<p>This is my first webpage.</p>
```

The comments help explain the code, but they will not be visible in the browser.

---

## Output

When you open the webpage, you will only see:

```text
HTML Comments

HTML comments help explain your code.

Comments are only visible in the HTML file and do not appear on the webpage.
```

The comments are hidden because browsers do not display them.

---

## Practice

Create a webpage with:

* One heading
* Two paragraphs
* A comment before each element

For example:

```html
<!-- Heading -->

<!-- First Paragraph -->

<!-- Second Paragraph -->
```

Save the file and open it in your browser. You'll notice that only the heading and paragraphs are displayed.

---

## What You Learned

After completing this lesson, you should know:

* What an HTML comment is.
* How to write a comment.
* Comments are not displayed on the webpage.
* Comments help make your code easier to read and understand.
