JavaScript Events

Learning how JavaScript responds to user actions.

What Is an Event?

An event is an action that happens inside the browser.

Examples include clicking, typing, moving the mouse, and submitting a form.

button.addEventListener("click", function () { console.log("Button clicked"); });

Common Browser Events

click

A user clicks an element.

dblclick

A user double-clicks an element.

mouseenter

The pointer enters an element.

mouseleave

The pointer leaves an element.

keydown

A keyboard key is pressed.

input

An input value changes.

Event Summary

Part Purpose
Element The selected HTML element
