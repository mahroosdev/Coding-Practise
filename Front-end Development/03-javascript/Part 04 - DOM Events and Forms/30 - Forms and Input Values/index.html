Reading Form Values

JavaScript uses the value property to read information entered into form fields.

const name = nameInput.value;

3. Form Properties

value

Reads the value of text fields and select elements.

checked

Checks whether a checkbox is selected.

preventDefault()

Stops the form from refreshing the page.

Form Event Summary

Property or Event Purpose
value Reads an input value
checked Reads a checkbox state
input Runs when an input value changes
submit Runs when a form is submitted
reset Runs when a form is reset
