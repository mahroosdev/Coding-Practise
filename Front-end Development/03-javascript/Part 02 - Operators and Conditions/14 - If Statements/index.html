JavaScript If Statements

Learning how JavaScript runs code only when a condition is true.

What Is an If Statement?

An if statement checks a condition.

The code inside the statement runs only when the condition is true.

if (condition) { // Code runs here }
JavaScript output will appear here.

1. A True Condition

When the condition is true, JavaScript runs the code inside the curly braces.

if (20 > 18)
JavaScript output will appear here.

2. A False Condition

When the condition is false, JavaScript skips the code inside the statement.

if (10 > 20)
JavaScript output will appear here.

3. Checking a Variable

Variables can be compared inside an if statement.

if (userAge >= minimumAge)
JavaScript output will appear here.

4. Checking Strict Equality

Strict equality can check whether a variable contains an expected value and data type.

if (courseName === "JavaScript")

5. Using Logical AND

The AND operator can require two conditions to be true.

if (hasEmail && hasPassword)

6. Using Logical OR

The OR operator runs the block when at least one condition is true.

if (isAdmin || isCreator)
