Important Switch Parts

switch

Contains the value being checked.

case

Defines one possible matching value.

break

Stops JavaScript from running the next case.

default

Runs when no case matches.

Switch Statement Summary

Keyword Purpose
switch Starts the value comparison
case Defines a possible match
break Stops the switch after a match
default Handles unmatched values
