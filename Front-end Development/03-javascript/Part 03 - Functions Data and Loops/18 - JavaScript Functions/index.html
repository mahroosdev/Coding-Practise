JavaScript Functions

What Is a Function?

A function is a reusable block of JavaScript code that performs a task.

function showMessage() { console.log("Hello from a function!"); }

1. Declaring a Function

A function declaration creates a named block of code.

2. Calling a Function

A function runs only when it is called.
