JavaScript Output

Learning different ways to display information with JavaScript.

What is JavaScript Output?

JavaScript output is information displayed by a JavaScript program.

The output can appear on the webpage, in the browser console, or inside a browser message box.

Output in the Browser Console

Open the browser console to see normal, warning, and error messages.

Normal message: JavaScript is working.

Warning message: This is a practice warning.

Error message: This is a practice error.

Output in an Alert Box

The alert() method can display a message inside a browser popup box.

alert("Hello from JavaScript!");

The alert example is included as practice because automatic popup boxes can interrupt the user.
