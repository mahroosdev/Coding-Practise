Introduction to JavaScript

Learning how JavaScript adds behaviour to webpages.

What is JavaScript?

JavaScript is a programming language used to make webpages interactive.

HTML creates the structure, CSS controls the design, and JavaScript controls the behaviour.

What Can JavaScript Do?

Change Content

JavaScript can change the text and other content on a webpage.

Respond to Users

JavaScript can respond when a user selects a button or enters information.

Perform Calculations

JavaScript can work with numbers and perform calculations.

Open the Browser Console
