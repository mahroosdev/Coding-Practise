CSS Variables

Reusing the same CSS values in different parts of a webpage.

What Are CSS Variables?

CSS variables allow us to save colors, spacing, and other CSS values so we can reuse them throughout the stylesheet.

Learn More

Why Use CSS Variables?

Reuse Values

The same variable can be used in many CSS rules without writing the original value again.

Easy Changes

Changing the value of one variable can update many elements on the webpage.

Cleaner CSS

Meaningful variable names can make the stylesheet easier to read and understand.

Simple Variable Example

The colors, spacing, border, and heading sizes on this page are controlled using CSS variables.

If a variable value is changed, every CSS rule using that variable will also use the new value.

Example

Variable declaration:

--primary-color: darkblue;

Using the variable:

color: var(--primary-color);
