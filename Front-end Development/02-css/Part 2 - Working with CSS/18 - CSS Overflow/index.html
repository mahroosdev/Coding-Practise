CSS Overflow

Visible Overflow

This box uses visible overflow. The content is longer than the available height, so it continues outside the box. This is the default overflow behavior.

Hidden Overflow

This box uses hidden overflow. The content is longer than the available height, but the extra content is hidden and cannot be seen outside the box.

Scroll Overflow

This box uses scroll overflow. The content is longer than the available height, so scrollbars are added. The user can scroll inside the box to read the remaining content.

Auto Overflow

This box uses auto overflow. Scrollbars appear only when the content is too large for the box. This is commonly used when content size may change.
