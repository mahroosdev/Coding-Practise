# 09 - HTML Horizontal Rules

## Introduction

Sometimes, you may want to separate different sections of a webpage.

In HTML, you can do this using the `<hr>` tag.

---

## What is a Horizontal Rule?

A horizontal rule creates a horizontal line on the webpage.

It is commonly used to separate different sections of content.

---

## Syntax

```html
<hr>
```

Place the `<hr>` tag where you want the horizontal line to appear.

---

## Example

```html
<h1>About Me</h1>

<p>I am learning HTML.</p>

<hr>

<h2>My Hobbies</h2>

<p>I enjoy coding and reading.</p>
```

The horizontal line separates the two sections.

---

## Output

The browser displays:

```text
My Portfolio

Welcome to my portfolio website.

----------------------------

About Me

I am learning HTML and building websites.

----------------------------

Contact

Thank you for visiting my webpage.
```

---

## Practice

Create a webpage with three sections:

* Home
* About
* Contact

Add a horizontal rule between each section.

---

## What You Learned

After completing this lesson, you should know:

* The `<hr>` tag creates a horizontal line.
* It is used to separate sections of content.
* The `<hr>` tag does not need a closing tag.
* It helps make a webpage easier to read.
